Full name model

Accuracy: 98%
Dataset: 630,795 Bengali full names

The name is directly passed to the model which then detects the gender of the entire name.

Word-wise model

Accuracy: 86%
Dataset: 61,664 Bengali single word names

The name is divided into individual words and the model detects the gender of each word. The dataset for this model was extracted from the dataset for full name model.
